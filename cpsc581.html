Project 1 - Voice Matcher

This project was making an interface that had audio as an input but no voice recognition. I created an app to see which person in a group was loudest or could match a decibel level. The inspiration for this project came from my younger brother and his friends shrieking and yelling all of the time; I decided to make an app to see which of them was the loudest.

Code from other sources: https://github.com/cwilso/volume-meter and Tutorials from CPSC 581 Winter 2021
